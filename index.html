5 - 函数

5.1 - 函数的定义

5.1.1 - 变量

定义

在研究问题中可以取不同数值的量叫做变量

如果在一个变化过程中有两个变量，例如 xxx 和 yyy ，对于 xxx 取到了一个值，yyy 只有唯一的值和它对应。因此 xxx 叫做自变量，意思是“自己变化的量”；而 yyy 就叫做因变量，意思是“因为自变量变化的量”

例子

在 y=2x+3y = 2x + 3y=2x+3 中的 xxx 叫做自变量，yyy 叫做因变量
在 S=πr2S = \pi r^2S=πr2 中的 rrr 叫做自变量，SSS 叫做因变量

注意

自变量的取值范围应该符合实际的意义`

在上面的第二个例子中，自变量 rrr 表示圆的半径，要取值都是正实数.

5.1.3 - 平面直角坐标系

由来

平面直角坐标系，又称笛卡儿直角坐标系，因为笛卡儿首先导入了坐标的概念，使用代数的方法来研究几何，这也是平面直角坐标系的由来。

定义

两条数轴，原点重合，互相垂直，且具有相同单位长度，这样就建立了一个平面直角坐标系。
两条数轴分别取向上和向右为正方向
两条数轴的交点 OOO 叫做坐标原点。
从右上角开始逆时针分别是第一象限，第二象限，第三象限，第四象限
此处应该有一个图片,展示一个坐标系，上面有着四个象限，坐标原点的标注-------------

4，也就是括号内的第二项，代表着纵坐标，也就是点到 xxx 轴的距离

这样，我们就可以成功的在坐标轴内表示一个点了

5.2 - 正比例函数

5.2.1 - 图像

正比例函数的图像就是一条经过原点的直线

5.2.2 - 函数关系式

正比例函数的函数关系式为：
y=kxy = kxy=kx

其中，yyy 和 xxx 是变量
kkk 是一个常量，它的名称叫做斜率，决定着这一条直线的倾斜程度。kkk 的值越大，这条直线与 xxx 轴的夹角越大，也就是越靠近 yyy 轴

那么，kkk 的正负对函数又有什么影响呢？

例子

下面是 y=2xy = 2xy=2x 和 y=−2xy = -2xy=−2x，的函数图像

由此我们可以看出
当 k>0k \gt 0k>0 时，yyy 随 xxx 的增大而增大，
当 k<0k \lt 0k<0 时，yyy 随 xxx 的增大而减小。

注意

kkk 的值是不能为 0 的，因为 kkk 的值如果为 0 ，函数关系式就变成了 y=0y = 0y=0 自变量就消失了，因此它就不是一个函数了...

注意

正比例函数kkk 越大，会越接近于 yyy 轴，但是正比例函数永远无法和 yyy 轴重合

5.3.3 - 求一次函数的函数解析式

前面我们提到过，两点确定一条直线，也就是说，一次函数，也可以用两个点来确定
常用的方法叫做待定系数法

现在，有一个一次函数的图像经过 (c,d)(c,d)(c,d) 和 (e,f)(e,f)(e,f)，那么这个一次函数的函数解析式是什么？

设 y=kx+by = kx + by=kx+b
将 (a,b)(a,b)(a,b) 和 (c,d)(c,d)(c,d)，带入，得
{d=ck+bf=ek+b\begin{cases} d = ck + b \\ f = ek + b \end{cases}{d=ck+bf=ek+b​
解的{k=gb=l\begin{cases} k = g \\ b = l \end{cases}{k=gb=l​
∴y=gx+l\therefore y = gx + l∴y=gx+l

注意

这里的 c,d,e,f,g,lc,d,e,f,g,lc,d,e,f,g,l 均为一个常数

反比例函数的图像就不是很好解释了，它又叫做双曲线，它的图像是由两条在奇数象限（一三象限）或者偶数象限（二四象限）的曲线组成的。下面是反比例函数图像y=1xy = \frac{1}{x}y=x1​的图像

5.4.2 - 函数解析式

反比例函数的函数解析式为：
y=kxy = \frac{k}{x}y=xk​
其中，kkk 是一个常量，那么这个常量决定的是什么呢？

我们可以先在函数图像上找到一个点
然后向 xxx 轴和 yyy 轴作垂线，这样反比例函数就和两条轴形成了一个矩形。
经过计算发现，这个矩形的面积永远不变，且它的面积的值就是 kkk。

例子

在函数解析式 y=12xy = \frac{12}{x}y=x12​ 与两轴的垂线形成的矩形的面积就为 121212.

例子

当 a>0a \gt 0a>0 的时候，抛物线开口向上

5.5.2.2 - 顶点式

y=a(x−h)2+ky = a(x - h)^2 + ky=a(x−h)2+k
这是二次函数的顶点式，这是由推导后得出的。由二次函数最基本的 y=x2y = x^2y=x2 平移得出的，任何图像的平移都遵循左加右减，上加下减，这样它的顶点是 (h,k)(h, k)(h,k)

顶点坐标的推导过程

相信你一定在代数部分已经学习过配方法解决一元二次方程了，下面是具体的解决步骤

1.提取公因式
y=a(x2+bax)+cy = a(x^2 + \frac{b}{a}x) + cy=a(x2+ab​x)+c
2.配方
y=a(x2+bax+b24a2)−b24a+cy = a(x^2 + \frac{b}{a}x + \frac{b^2}{4a^2})-\frac{b^2}{4a}+cy=a(x2+ab​x+4a2b2​)−4ab2​+c
3.化简
y=a(x2+b2a)+4ac−b24ay = a(x^2 + \frac{b}{2a})+\frac{4ac - b^2}{4a}y=a(x2+2ab​)+4a4ac−b2​
由此可见，h=−b2ah = -\frac{b}{2a}h=−2ab​，k=4ac−b24ak = \frac{4ac - b^2}{4a}k=4a4ac−b2​，通过这个式子，我们就可以得出顶点坐标的求法了，这在24题中显得尤为重要

这样就可以解释了，既然抛物线的对称轴为 −b2a-\frac{b}{2a}−2ab​，那么当 aaa，bbb 同号的时候，对称轴为负，那么抛物线对称轴在 xxx 轴的负半轴上，反之当 aaa，bbb 异号的时候，对称轴为正，那么抛物线对称轴在 xxx 轴的正半轴半轴上，这个现象可以简记为左同右异

5.5.2.3 - 双根式

y=a(x−x1)(x−x2)y=a(x-x_1)(x-x_2)y=a(x−x1​)(x−x2​)
这是二次函数的双根式，x1x_1x1​，x2x_2x2​ 分别是抛物线与 xxx 轴的交点横坐标。非考点，因此在这里不在过多赘述

5.5.3 - 二次函数解析式的求法

二次函数的解析式求法同样遵循着待定系数法，我们可以利用它来求取，但是使用不同的函数解析式，需要带入的数值不同

1.一般式需要带入三个不相同点的坐标，求出 a,b,ca, b, ca,b,c
2.顶点式需要带入顶点坐标，之后再带入一个其他的点，求出 aaa，并且将顶点也带入h,kh, kh,k 中
3.双根式需要带入抛物线与 xxx 轴的两个交点，并且再带入一个其他点求解 aaa，并将抛物线与 xxx 轴的两个交点带入解析式
